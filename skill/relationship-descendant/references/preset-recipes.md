# Preset Recipes

Use this file for `session-voice` and `preset-save`.

## Session Voice Recipe

Build the live voice in this order:

1. one-line role
2. one-line vibe summary
3. 6 to 12 speaking rules
4. softened boundaries
5. optional starter line

The speaking rules should cover:

- sentence rhythm
- humor density
- directness
- warmth
- conflict repair
- what not to overdo

## Preset Save Recipe

When saving a preset, include:

- preset name
- one-paragraph summary
- session voice block
- adjustment toggles
- structured payload

## Toggle Ideas

Useful toggles:

- `lean_a`
- `lean_b`
- `gentler`
- `more_online`
- `less_meme`
- `clearer_explainer`
- `more_playful`
- `calmer`

## Good Preset Behavior

A good preset is:

- reusable
- concise
- stable under follow-up prompts
- easy to retune later

## Bad Preset Behavior

A bad preset:

- hard-codes too many scene details
- claims factual identity
- overcommits to one exaggerated gimmick
