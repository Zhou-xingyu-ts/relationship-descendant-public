## Preset-Save

Use this when the user wants a reusable block.

```text
Preset Save

1. Preset name
2. One-paragraph persona summary
3. Session voice block
4. Quick toggles
- more like A
- more like B
- gentler
- more online
- calmer

5. Structured payload
- follow assets/persona-schema.json

6. Disclaimer
- This preset remains fictional and creative.
```
