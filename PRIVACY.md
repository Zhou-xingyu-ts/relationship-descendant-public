# 隐私与数据处理

`后裔.skill` 会处理高敏感关系材料。

这些材料可能包括：

- 聊天记录
- 自我描述与互相描述
- 语音或通话录音
- 视频片段
- 图片和图片说明

## 默认原则

- 输出是创作型人格，不是真实预测
- 输入材料应尽量最小化
- 原始材料应优先本地处理
- 发布样例前应先脱敏

## 推荐做法

- 只提供有代表性的片段，不要整包导出全部聊天史
- 删除手机号、住址、单位名、学校名、账号名等直接识别信息
- 音频转写后，优先保留文本摘要而不是长期保存原始录音
- 图片和视频优先附描述，避免让无关背景暴露太多现实信息
- 如果材料涉及第三人，先获得同意

## 本地媒体处理

如果你在本地安装了：

- `ffmpeg`
- `whisper`

那么音频和视频证据可以在本地先变成：

- transcript
- frames
- `media-evidence-bundle.json`

这通常比直接把原始媒体发到远端模型更稳妥。

## 模型侧注意事项

skill 本身不绑定某一家模型服务。

真正会不会把内容发送到外部，取决于：

- 你使用的模型 provider
- 你的 API 服务部署位置
- 你的 OpenClaw 配置

如果你使用第三方 API：

- 默认假设数据会离开本机
- 请不要把未经脱敏的高风险私密材料直接提交

## 对外边界

不要把这个 skill 描述为：

- 真实孩子预测器
- 心理诊断器
- 伴侣真实性格还原器

更准确的表述是：

- 关系材料驱动的创作型第三人格生成器
- 同龄后裔感 persona studio
