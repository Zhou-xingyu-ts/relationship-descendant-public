# Release Assets Checklist

Use this before publishing to GitHub or a public skill marketplace.

## Minimum

- polished README
- install script
- smoke test script
- one-line repo description
- three example prompts
- one sample output
- safety line
- privacy line
- one ready-to-use banner image
- one ready-to-use social card

## Better

- one square cover image
- one landscape banner
- one short demo GIF
- one screenshot of a clean output

## Best

- banner image
- GIF of the workflow
- comparison image showing portrait vs blend-lab vs session-voice
- pinned launch post copy
