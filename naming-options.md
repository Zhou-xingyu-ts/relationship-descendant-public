# Naming Options

这版改成了更接近 README 海报风格的命名。

## Recommended Default

### 后裔.skill

Why it works:

- 一眼就有“公开 skill 项目”的味道
- 比 `relationship-descendant` 更像一个会传播的名字
- 有你给的参考图那种“标题就是概念”的感觉

Best for:

- README 主标题
- 社交传播
- 截图封面

## Secondary English Name

### Relationship Descendant Studio

适合：

- 仓库副标题
- 英文说明
- 国际用户解释

## Strong Alternatives

### 关系后裔.skill

Pros:

- 更直白
- 中文语义更完整

Cons:

- 没有 `后裔.skill` 那么干净

### 第三声.skill

Pros:

- 更文学化
- 更强调“关系长出第三个声音”

Cons:

- 不够直观

### 后代感.skill

Pros:

- 很贴近玩法

Cons:

- 稍微偏网名感
