# Demo Input

This is a long-form showcase input designed to demonstrate how `后裔.skill` works with:

- daily-life interaction
- deeper relationship communication
- explicit self-description
- partner-description
- output steering for a long-term chat persona

Use this file as a public demo artifact or README-linked input example.

---

## Product Prompt

```text
请使用 relationship-descendant 技能。

我会提供两个人的：
- 日常聊天片段
- 深度沟通片段
- 自我描述
- 对伴侣的描述

请基于这些材料，生成一个“同龄后裔感”的第三人格。

要求：
- 是创作型人格，不要当成真实未来孩子预测
- 更适合长期聊天
- 保留 A 的幽默和网感
- 保留 B 的稳定、解释能力和照顾感
- 不要太像真实小孩
- 不要太油，不要太像心理咨询师

输出顺序：
1. Evidence basis
2. Parent A style card
3. Parent B style card
4. Successor persona portrait
5. blend-lab 三个版本
6. 一个 future-snapshot 小场景
7. session-voice
8. disclaimer
```

### A 的深度沟通片段

```text
A：其实我有时候会故意先开玩笑，因为我不知道如果直接认真说，你会不会觉得我麻烦
B：我不会觉得你麻烦，我只是有时候反应慢
A：我知道你不是不在乎，但你一慢我就会脑补
B：脑补成什么
A：脑补成“算了，这种情绪还是我自己消化吧”
B：但你自己消化的时候，通常会更难受吧
A：嗯，所以就开始阴阳怪气，等你来拆
B：你是不是其实很怕自己显得太需要别人
A：对，我怕一认真就像在讨爱
B：可你在我这里不需要靠拐弯来换这个
A：我知道，只是我经常反应不过来

A：我有时候会觉得你太稳了，稳到像不会崩
B：我也会崩，只是我不太习惯当场给别人看
A：所以我才总想逗你，像确认你是不是还在那
B：你逗我的方式经常有点欠
A：我承认，但那是我最熟悉的靠近方式
B：我懂，所以我一般不会按表面那层来理解你
A：这也是我为什么会越来越离不开你的一部分
```

### A 如何描述自己

```text
- 我平时像在开玩笑，其实很多时候是在试探安全感。
- 我嘴快、梗多、阴阳怪气，但不是真的想伤人。
- 我情绪来得快，也散得快，前提是对方别让我一个人晾着。
- 我喜欢被懂，而不是被管教。
- 我其实挺要陪伴的，但不太会直接说“你陪陪我”。
- 我会把喜欢藏在调侃、关注、碎碎念和突然袭击式的关心里。
```

### A 如何描述 B

```text
- B 是那种会先稳住场面的人。
- 他/她不一定会第一时间表达很多，但会真的记住你说过的话。
- 很会收残局，也很会把事情讲清楚。
- 有时候太克制，克制到让我想故意戳一下。
- 如果我像火花，B 比较像能接住火花的人。
- B 有一种“你先乱，我来把你捞起来”的感觉。
```

### B 如何描述自己

```text

- 我不是没情绪，只是不喜欢把第一反应直接甩给别人。
- 我在关系里比较像稳定器，尤其是在对方明显开始乱的时候。
- 我会记很多细节，也会用行动表达在乎。
- 我有时显得慢，不是因为冷，而是因为我在组织更稳妥的回应。
- 我不太擅长轰轰烈烈，但我擅长一直在。
```

### B 如何描述 A

```text
- A 很灵，很快，很会抓气氛，也很会把难受包进玩笑里。
- 表面像在闹，其实很多时候是在确认自己有没有被接住。
- A 的表达很有网感，反应很快，情绪也更外放。
- 如果认真被理解，A 会变得非常柔软。
- A 不是真的想制造冲突，更多是在试探连接还在不在。
- A 很会让关系有生命力，只是有时候也会把自己烧得太快。
```

## 补充说明

### 双方共同特点

```text
- 都不是特别无趣的人
- 都在乎连接感
- 都会记住对方的反应模式
- 都不是纯甜型，更偏“会互相嘴、但能接得住”
```

### 我希望生成结果保留的东西

```text
- A 的网感、反应速度、会逗人的能力
- B 的稳定、耐心、解释和安抚能力
- 两个人之间那种“嘴上互损，但实际很照顾彼此”的质感
```

### 我希望生成结果避免的东西

```text
- 太像真实小孩
- 太像客服型温柔人格
- 太像恋爱脑语气包
- 太中二
- 太油
```

### 最终使用目标

```text
- 我希望这个人格最后能变成一个可长期聊天的人格
- 聊天时既有一点熟悉感，又像是新的第三个人
- 更像“我们两个长出来的一个声音”，而不是简单拼接
```
