# Example Prompts

These are written in a more public "store prompt" style.

## 1. First-Time Portrait

```text
Use relationship-descendant.
Based on the following chats and notes, create a fictional same-age successor persona.
Keep it creative, not predictive.

Please output:
1. Parent A style card
2. Parent B style card
3. Successor persona portrait
4. Inherited traits
5. One-line disclaimer
```

## 2. Variant Shopping

```text
Use relationship-descendant in blend-lab mode.
Give me 3 versions of the same fictional successor persona:
- A-heavy
- balanced
- gentler and more stable

Keep the humor, reduce harshness, and make all 3 suitable for long-term chat.
```

## 3. Show Me How They Sound

```text
Use relationship-descendant in future-snapshot mode.
First define the main persona, then write 2 short same-age scenes showing how this third persona speaks in practice.
Keep it emotionally readable and creative, not predictive.
```

## 4. Turn It Into A Chat Voice

```text
Use relationship-descendant.
Create the persona, then convert the selected version into a session-voice block for long-term chat.
Keep the humor, reduce aggression, and do not make it feel like a real child.
```

## 5. Preset Export

```text
Use relationship-descendant.
Build a fictional same-age successor persona from these materials, then output:
- portrait
- best recommended version
- session-voice
- preset-save
```
